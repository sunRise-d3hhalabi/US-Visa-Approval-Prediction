## Git commands

```bash
git add .

git commit -m "Updated"

```

## How to run?

```bash
conda create -n visa python=3.8 -y
```

```bash
conda activate visa
```

```bash
pip install -r requirements.txt
```

## Dataset

```bash
Dataset: EasyVisa_Dataset (Kaggle)

https://www.kaggle.com/datasets/moro23/easyvisa-dataset/data
```
